NovaFit

> Personalized fitness app that calculates your exact diet and workout routines based on your body metrics, preferences, and goals.
About

NovaFit is a complete fitness platform built with **Kotlin (Android)** and **React (Web Portal)**. It uses a custom AI-powered algorithm to generate personalized nutrition plans and workout routines for each user based on their physical data.

---
Features

Android App (Kotlin)
- Smart Body Calculator— calculates exact caloric needs based on weight, age, measurements, and gender
- Personalized Diet— generates custom meal plans based on your food preferences
- Custom Workout Routines— tailored exercise plans with step-by-step image guides
- Animated Coach— custom character that guides you through each exercise
- Real-time Sync— all data syncs instantly via Firebase
- User Profile— edit your info, track your progress and stats

Web Admin Portal (React)
- 👥 User Management — view and manage registered users
- 🏃 Exercise Management — add/remove exercises with images
- 🥦 Nutrition Database — manage proteins and food items
- 📈 tatistics Dashboard — view platform-wide stats
